# Not App

Cross-platform note-taking application developed with Flask, React.js, and React Native.

## Features

- User registration and login
- JWT-based authentication
- Create, list, update, and delete notes
- Web client with React.js
- Mobile client with React Native / Expo
- SQLite database integration

## Technologies

### Backend
- Python
- Flask
- SQLite
- JWT Authentication
- Flask-CORS

### Web Client
- React.js
- Material UI
- Axios
- React Router

### Mobile Client
- React Native
- Expo
- AsyncStorage
- Axios
- Expo Router
